TEL-Audiobook-Ripper by turnerendlesslearning

A Java application that uses cdparanoia and oggenc to quickly rip audiobook cds and edit track names and info for proper sorting.

This program is written in Java and designed to be used on a Linux system. The program employs system commands to rip cds using cdparanoia and convert them to the ogg format using oggenc (part of vorbis-tools). The resulting files are organized by book name with subfolders for the cd number. The program uses the preferences API to remember the book name and author and to autoincrement the CD number to save time.

Depdendencies

Must have cdparanoia and vorbis-tools installed.

Install

The repository contains the entire Netbeans 7.1 project. You can open it with Netbeans and build and run it that way, or compile the Java as you normally would.
